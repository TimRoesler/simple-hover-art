# Simple Hover Art

A lightweight Foundry VTT module that displays an actor's portrait or token art while the user hovers over a token.

## Features

- Displays actor portrait or token art, with video support.
- Configurable delay, card size, caption, and screen position.
- Optional visibility requirements: none, observer, or owner.
- Optional NPC visibility restriction for non-GM users.
- English and German localization.

## Compatibility

- Foundry VTT 13+
- Verified for Foundry VTT 14

## Installation

In Foundry VTT, open **Add-on Modules → Install Module** and paste:

```text
https://github.com/TimRoesler/simple-hover-art/releases/latest/download/module.json
```

Then activate **Simple Hover Art** in the desired world.

## Configuration

Configure the module under **Game Settings → Configure Settings → Simple Hover Art**.
